本地环境

PierCode 把 ChatGPT、Claude、Gemini、Qwen、Kimi 连接到你的文件系统和浏览器。AI 输出工具调用，Chrome 扩展代理到本地沙箱服务器——读文件、改代码、跑命令、操作浏览器，全程经你批准。

开始使用 阅读文档
无需复制粘贴。不上传云端。运行在 127.0.0.1.
piercode — zsh

工作原理

每次工具调用，四个步骤。

  1. 1

    AI 输出工具块

    助手在回复里输出一个可见的 piercode-tool JSON 代码块。

  2. 2

    扩展检测到它

    Chrome 扩展识别代码块并在页面上弹出批准卡。

  3. 3

    本地服务器执行

    127.0.0.1 上的 Go 服务器校验后在工作目录沙箱内执行。

  4. 4

    结果回填

    输出流回对话——AI 拿到真实的本地结果继续工作。

本地优先的设计

AI 能碰什么，由你决定。

仅回环地址

服务器绑定 127.0.0.1，绝不暴露公网端口。

⚷

Token 认证

每次启动生成 token，存于 ~/.piercode，每个请求都要带。

▣

路径沙箱

路径经真实路径解析校验，始终留在工作目录内。

快速开始

三步拥有本地 AI 智能体。

1

启动服务器

go run ./cmd/server -dir .
2

构建扩展

cd extension && npm install && npm run build
3

在 Chrome 加载

chrome://extensions → Load unpacked → extension/dist
