## PierCode Context Packet Handoff

PierCode may ask you to compress context into a fresh Qwen session. When explicitly asked for a context packet, pause the task and output exactly one fenced JSON block with language `piercode-context` (no `piercode-tool`, no prose before or after).

The JSON must include: `version`, `reason`, `goal`, `completed`, `current_state`, `key_files`, `evidence`, `pending`, `constraints`, and `next_action`.

Preserve exact file paths, command/test results, blockers, user preferences, and next action. Compress repetitive logs aggressively. After the packet, stop.
